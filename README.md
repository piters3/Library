# Library

Library management system created in ASP.NET MVC.

### Prerequisites

This project is build with technologies:

* ASP.NET MVC 5
* Entity Framework 6
* ASP.NET Identity 2
* ASP.NET SignalR 2
* Visual Studio 2017 Community
* MS SQL Server 2016 LocalDB
* jQuery 3
* Bootstrap 3.3.7
* font-awesome 4.7
* pace.min.js 1.0.2
* dataTables 1.10.10
